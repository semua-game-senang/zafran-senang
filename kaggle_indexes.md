### ⚡ TL;DR for the competition

For this specific competition the text processing you actually need is just:

```python
import re

def clean(text):
    text = str(text).lower()
    text = re.sub(r'http\S+', '', text)        # remove URLs
    text = re.sub(r'[^a-z0-9\s]', ' ', text)  # remove punctuation
    text = re.sub(r'\s+', ' ', text).strip()   # normalize spaces
    return text
```
